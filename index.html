Flash plugin not found - follow the steps below to set it up

Step 1: Get the Pepper Flash Plugin

macOS Check if you have it from old Chrome: /Library/Internet Plug-Ins/PepperFlashPlayer/PepperFlashPlayer.plugin Or extract from Chrome 87 or earlier: /Applications/Google Chrome.app/Contents/Versions/*/Google Chrome Framework.framework/Internet Plug-Ins/PepperFlashPlayer.plugin
Windows Check system directory: C:\Windows\System32\Macromed\Flash\pepflashplayer64_32_0_0_465.dll Or extract from old Chrome installation.
Linux Search for: libpepflashplayer.so
Can't find it? Search the Internet Archive for "Pepper Flash 32.0.0.465" for your platform.

Step 2: Copy to Plugins Folder

Copy the plugin file to the plugins folder next to this app: plugins/PepperFlashPlayer.plugin (Mac) plugins/pepflashplayer64.dll (Windows) plugins/libpepflashplayer.so (Linux)

Step 3: Restart the App

Close and reopen Flash Player. The game should load automatically!
Game URL: By default, the player connects to https://example.com/flash-game. You can change this by setting the GAME_URL environment variable.
